Databases served by this provider

  • Google DeepMind's Gnome Dataset (gnome)

    https://optimade-gnome.odbx.science

    An OPTIMADE API for the recent dataset released by Google DeepMind containing 340k new on-hull materials, according to PBE.

    Properties served by this database:

    By entry type (click to expand):
    • references
      • The full list of standard OPTIMADE properties.
    • structures
      • The full list of standard OPTIMADE properties.
      • _gnome_bandgap Unit: eV
        Description: The band gap computed with PBE in eV
        Type: float
      • _gnome_decomposition_energy_per_atom Unit: eV
        Description: Decomposition energy in eV per atom, relative to stable phases on the convex hull of known phases(excluding the GNome dataset).
        Type: float
      • _gnome_formation_energy_per_atom Unit: eV
        Description: Formation energy per atom, computed versus elemental references with PBE
        Type: float
      • _gnome_space_group_it_number Description: The IT number of the space group of the structure
        Type: integer
      • _gnome_space_group_symbol_hermann_mauguin Description: The HM symbol of the space group of the structure
        Type: string
